# Safe Student Delivery
## Prerequisites
- xampp with Apache and SQL server running

## Run
Copy this repository's contents into <xampp_installation_location>/htdocs/.

The new location should be 
- /Applications/XAMPP/htdocs/repository_name
- C:\Program_Files(x86)\XAMPP\htdocs\repository_name

Open login.php in a browser at URL localhost/repository_name/login.php.
